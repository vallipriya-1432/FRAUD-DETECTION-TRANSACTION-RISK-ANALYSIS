## üìä Key Insights

- Fraud accounts for less than 0.2% of total transactions, creating a class imbalance challenge.  
- The Logistic Regression model performed well with strong recall and AUC scores.

- The model can serve as a real-time fraud alert system, flagging suspicious activity for review.
